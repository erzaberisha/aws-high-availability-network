# master-webapp

AWS High-Availability & Secure Infrastructure Project
- Technologies used: AWS (VPC, EC2, ALB, ASG, NAT Gateway), Git, Apache (httpd), Networking (TCP/IP, Subnetting).
- Designed and deployed a highly available, secure, and self-healing hybrid network architecture on AWS.
- Configured a custom VPC with 6 subnets (public and private), NAT Gateways, Route Tables, and strict Security Groups to isolate backend servers from the public internet.
- Implemented an Application Load Balancer (ALB) and Auto Scaling Group (ASG) to automate traffic distribution and achieve a self-healing infrastructure.
- Utilized AWS User Data scripts to automate server provisioning, Apache web server installation, and CI/CD code deployment via Git.
